Kaskadni stilovi

Postoje 3 vrste primene kaskadnih stilova ha html dokumentima.

  1. Direktna primena stilova kkao atribyut nekog taga koji zelimo da menjamo.
  2. Unutar head taga imamo style tag unutar kog vrsimo primenu CSS.
  3. Eksterni CSS. Njega zapravi karakterise novi fajl (npr. style.css)
